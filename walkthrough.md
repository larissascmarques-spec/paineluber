# Walkthrough: Painel Dinheiro Mudo

O sistema está pronto! Abaixo, sigo o passo a passo para você testar a solução agora mesmo no seu computador ou tablet.

## 1. Como Usar o Dashboard (Cofre do Motorista)
1. **Abra o arquivo [index.html](file:///c:/Users/LARISSA/Desktop/extension%20UEDA/painel-dinheiro-mudo/index.html)** no seu navegador.
2. Você verá o **Painel de Controle** com 6 slots de anúncios vazios.
3. Clique em qualquer slot para abrir o editor.
4. **Suba as fotos dos anúncios** que eu gerei para você (veja na galeria abaixo).
5. Defina o tempo de exibição (ex: 10 segundos).
6. Salve cada slot. Os cartões ficarão com uma borda roxa e o ícone de "check" verde ✅.

## 2. Ativando o Painel no Tablet
- Clique no botão **🚀 INICIAR PAINEL NO TABLET**.
- O sistema entrará em modo tela cheia (fullscreen) e começará o loop automático dos anúncios ativos.
- Uma **barra de progresso** sutil na parte inferior indica quanto tempo falta para o próximo anúncio.
- Para sair do modo player, basta apertar a tecla **ESC** ou clicar no "X" no canto superior direito.

## 3. Script de Vendas (Bônus)
O botão **"Copiar Script de Vendas"** no dashboard já vem com o texto que eu preparei para o motorista enviar via WhatsApp para comerciantes locais. Ele é focado em escassez ("último slot") e prova social ("já tenho um dentista e uma barbearia").
